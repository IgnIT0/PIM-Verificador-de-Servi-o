Sistema desenvolvido em Python para acompanhamento de ativos de rede e serviços em tempo real.

-Monitoramento de conectividade (Ping e HTTP).
-Interface gráfica com visualização em lista.
-Registro de logs com data formatada (DD/MM/AAAA).
-Tratamento automático de URLs para evitar conflitos de prefixo.
-------------------------------------------------------------------------------------------------------------------------------------------------------------
Como usar:

Instalação: Certifique-se de ter o Python instalado e as dependências citadas acima.

Execução: Execute o arquivo principal (ex: python main.py).

Configuração: Insira o IP ou URL do dispositivo que deseja monitorar no campo de entrada.

Monitoramento: Clique no botão para iniciar a verificação. O status (Online/Offline) aparecerá na tabela central e os eventos serão registrados no log.
-------------------------------------------------------------------------------------------------------------------------------------------------------------
Requisitos:

Python 3.x
Bibliotecas: pandas, requests e tkinter.
